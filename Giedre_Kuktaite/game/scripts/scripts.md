These are the godot scripts necessary to run the game. All of these should be loaded into godot and assigned to the corresponding characters.
"Player" is the script illustrating character movement, death and win conditions
"bee" the script used for moving enemies-bees and hedgehogs
"hazzards" script used for hazzardous tiles-water
"movingplatform_" up-for up and down platform, side-for side to side moving platfor
"snake"-script used for snake
"baby"-game victory object script
